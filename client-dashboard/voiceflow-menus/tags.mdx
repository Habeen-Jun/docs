## Overview

Voiceflow Knowledge Base Tags is a feature **(not a menu)** in the Knowledge Base menu that allows clients to manage tags for the knowledge base data sources.
You can enable/disable the tags for the knowledge base data sources in the client access settings.

    ![client-access-tags](../images/client-access-tags.png)


Once you enable the tags feature for the project, you will see a bit different UI in the Knowledge Base menu.

    ![knowledge-base-tags](../images/kb-with-tags.png)

You can perform the following actions in the Knowledge Base Tags menu:

- View Tags: View the tags assigned to the knowledge base data sources.
- Add Tags: Add new tags to the knowledge base data sources.
- Edit Tags: Edit the existing tags assigned to the knowledge base data sources.
- Delete Tags: Delete the tags assigned to the knowledge base data sources.


## Creating Tags

1. Sign in to the ChatDash client dashboard.
2. Click on the Knowledge Base menu.
3. Type the tag name in the search bar and hit enter.
4. You will see the tags in the right side of the search bar.

    ![tags](../images/tag-search-bar.png)


## Deleting Tags

1. Sign in to the ChatDash client dashboard.
2. Click on the Knowledge Base menu.
3. Click on the `x` icon of the tag in the search bar to delete the tag.


## Assigning Tags to Knowledge Base Data Sources

1. Sign in to the ChatDash client dashboard.
2. Click on the Knowledge Base menu.
3. Click on the `+` icon in the tags column of the knowledge base data sources.

    ![assign-tags](../images/assign-tags.png)

4. Click on the tags you want to assign to the knowledge base data sources.
5. You will see the tags assigned to the knowledge base data sources.

    ![assigned-tags](../images/assigned-tags.png)


## Detaching Tags from Knowledge Base Data Sources

1. Sign in to the ChatDash client dashboard.
2. Click on the Knowledge Base menu.
3. Click on the `x` icon of the tag you want to detach from the knowledge base data sources.
